# Conciliador BROU

Herramienta para conciliación bancaria entre SAP y BROU en:

* USD
* UYU
* EUR

## Uso

1. Abrir `index.html`
2. Cargar archivo SAP
3. Cargar archivo Banco
4. Ejecutar conciliación (requiere backend o ejecución local)

## Scripts

* `SCRIPT_USD_BROU.py`
* `SCRIPT_UYU_BROU.py`
* `SCRIPT_EUR_BROU.py`

## Notas

* GitHub Pages solo muestra la interfaz
* Para ejecutar conciliaciones se requiere entorno Python
